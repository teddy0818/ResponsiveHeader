# HTML

- html 은 브라우저에서 실행 가능한 가장 기본적인 파일이다
- head 부분은 특징을 정의하는 영역 / body 부분은 실제로 보여지는 영역
- MDN 사이트에서 태그나 호환성에 대한 정보를 얻을 수 있다
